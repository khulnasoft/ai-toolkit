# Jest: cannot find module '@ai-toolkit/rsc'

## Issue

I am using AI TOOLKIT RSC and am writing tests for my RSC components with Jest.

I am getting the following error: `Cannot find module '@ai-toolkit/rsc'`.

## Solution

Configure the module resolution via `jest config update` in `moduleNameMapper`:

```json filename="jest.config.js"
"moduleNameMapper": {
  "^@ai-toolkit/rsc$": "<rootDir>/node_modules/@ai-toolkit/rsc/dist"
}
```
