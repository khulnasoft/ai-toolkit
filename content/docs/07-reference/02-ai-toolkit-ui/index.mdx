# AI TOOLKIT UI

[AI TOOLKIT UI](/docs/ai-toolkit-ui) is designed to help you build interactive chat, completion, and assistant applications with ease.
It is framework-agnostic toolkit, streamlining the integration of advanced AI functionalities into your applications.

AI TOOLKIT UI contains the following hooks:

## UI Framework Support

AI TOOLKIT UI supports the following frameworks: [React](https://react.dev/), [Svelte](https://svelte.dev/), [Vue.js](https://vuejs.org/),
[Angular](https://angular.dev/), and [SolidJS](https://www.solidjs.com/).

Here is a comparison of the supported functions across these frameworks:

|                                                                     | [useChat](/docs/reference/ai-toolkit-ui/use-chat) | [useCompletion](/docs/reference/ai-toolkit-ui/use-completion) | [useObject](/docs/reference/ai-toolkit-ui/use-object) |
| ------------------------------------------------------------------- | ------------------------------------------------- | ------------------------------------------------------------- | ----------------------------------------------------- |
| React `@ai-toolkit/react`                                           | <Check size={18} />                               | <Check size={18} />                                           | <Check size={18} />                                   |
| Vue.js `@ai-toolkit/vue`                                            | <Check size={18} />                               | <Check size={18} />                                           | <Check size={18} />                                   |
| Svelte `@ai-toolkit/svelte`                                         | <Check size={18} /> Chat                          | <Check size={18} /> Completion                                | <Check size={18} /> StructuredObject                  |
| Angular `@ai-toolkit/angular`                                       | <Check size={18} /> Chat                          | <Check size={18} /> Completion                                | <Check size={18} /> StructuredObject                  |
| [SolidJS](https://github.com/kodehort/ai-toolkit-solid) (community) | <Check size={18} />                               | <Check size={18} />                                           | <Check size={18} />                                   |
